Privacy Policy — Free Dark Mode

Last updated: 2026-04-14

Free Dark Mode is a local-only browser extension. It does not send any browsing data, page contents, or personal information to external servers. All settings are stored locally in the user's browser via `chrome.storage.local`.

What data is stored:
- User preferences and theme settings (brightness, contrast, color values) stored locally.
- Optional per-site override list stored locally.

What data is NOT collected or transmitted:
- Page content, browsing history, cookies, or user files.
- No telemetry or analytics are included.

Contact:
If you have questions, create an issue or contact the repository owner.
